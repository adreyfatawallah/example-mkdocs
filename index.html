This is a simple design and documentation API Monad

General Token

Get general access token to access API

POST

http://127.0.0.1:8000/general-token

Field Type Description Request
client_secret String Client secret from monad Header 
{
    "diagnostic": {
        "status":200,
        "description":"success"
    },
    "response": {
        "access_token":"khgsdhgsdkjfgiojijigjsfkjgildfjgfildgjl",
    }
}

Login

POST

http://127.0.0.1:8000/login

Field Type Description Request
access_token String Access token from general token Header
username String Username Header
password String Password Header 
{
    "diagnostic":{
        "status":200,
        "description":"success"
    },
    "response":{
        "type":"Bearer",
        "access_token":"JfdjkfhIIDFDbfdjsdfs",
    }
}

Sending Money

Tranfer money to other user

check receiver

GET

http://127.0.0.1:8000/check-receiver

Field Type Description Request
Authorization String Access token login Header
user_id String User id receiver Header
nominal String Nominal Transfer Header 
{
    {
    "diagnostic":{
        "status":200,
        "description":"success"
    },
    "response":{
        "name_receiver":"User"
        "nominal_transfer":"1500000"
    }
}
